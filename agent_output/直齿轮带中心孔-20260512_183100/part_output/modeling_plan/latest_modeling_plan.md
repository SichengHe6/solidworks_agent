## 建模思路：直齿轮（模数2，齿数24，带12mm中心孔）

该零件将基于标准直齿轮模板进行参数化驱动，并添加二次切除特征以满足中心孔需求。

### 1. 基础资产与基准
- **标准件源**：`D:\a_src\python\sw_agent\standard_swpart\gear\spur_gear.SLDPRT`
- **工作副本**：将源文件物化至工作区，命名为 `spur_gear_m2_z24_h12.SLDPRT`。
- **主基准**：
  - **XY平面**：齿轮的底面（Z=0），用于绘制中心孔草图。
  - **原点**：齿轮几何中心及旋转中心。

### 2. 特征顺序
1.  **参数驱动（主动变量修改）**：
    - 修改 `M` (模数) = 2
    - 修改 `Z` (齿数) = 24
    - 修改 `B` (齿宽) = 10
    - 修改 `Alpha` (压力角) = 20（保持默认）
2.  **中心孔草图**：在 `XY` 平面上创建一个以原点为中心的圆，直径为 `12mm`（半径 `0.006m`）。
3.  **拉伸切除**：执行 `extrude_cut`，深度设为齿宽 `10mm`（`0.01m`），实现完全贯穿。
4.  **参考几何创建**：
    - 创建穿过原点且垂直于 `XY` 平面的参考轴。
    - 创建位于齿轮顶端面和中面的参考平面。

### 3. 关键接口命名
为便于后续装配引用，将显式创建并命名以下接口：
- **轴线接口**：`shaft_axis`（中心旋转轴）。
- **面接口**：
  - `front_face`：齿轮顶端面（Z = 0.01m 处）。
  - `back_face`：齿轮底端面（Z = 0 处）。
  - `bore_cylindrical_face`：中心孔的内圆柱面。
  - `gear_mid_plane`：齿轮中面（Z = 0.005m 处）。

### 4. 保存目标
- **模型文件**：`D:\a_src\python\sw_agent\agent_output\直齿轮带中心孔-20260512_183100\part_output\code\spur_gear_m2_z24_h12.SLDPRT`

### 5. 风险点说明
- **单位换算**：所有输入（mm）必须在代码中转换为米（m）。例如，12mm -> 0.012m。
- **特征依赖**：必须先通过 `set_global_variable` 更新齿轮主体尺寸，再进行二次切除，否则切除特征可能因主体直径过小或过大而定位失效。
- **API调用限制**：二次切除需严格遵循已验证的封装流程（`insert_sketch_on_plane` -> `create_circle` -> `extrude_cut`），避免使用不稳定的底层 COM 选择接口。
- **接口稳定性**：确保 `shaft_axis` 的定义点不随几何变化而丢失。
