## 太阳轮 (Sun Gear) 建模思路

该零件为行星齿轮减速机构的核心输入组件，将基于标准齿轮模板进行物化修改。

### 1. 特征顺序规划
1.  **物化标准件**：从标准件库 `spur_gear.SLDPRT` 复制并重命名为 `sun_gear.SLDPRT` 到工作目录。
2.  **参数驱动修改**：通过全局变量修改齿轮的基本几何参数（模数、齿数、齿宽）。
3.  **中心通孔切除**：在齿轮中心创建一个直径 15mm 的贯穿通孔。
4.  **参考几何体创建**：创建用于装配的基准面（中面、端面）和基准轴（中心轴）。

### 2. 关键草图与参数
*   **主动全局变量**：
    *   `M` (模数): 2mm
    *   `Z` (齿数): 18
    *   `B` (齿宽): 20mm
*   **中心孔草图**：在 `XY` 平面（齿轮底面）绘制一个直径为 15mm ($0.015\text{m}$) 的圆，圆心位于原点 $(0,0)$。
*   **切除特征**：使用 `extrude_cut` 向上切除，深度为齿宽 20mm ($0.02\text{m}$)。

### 3. 基准与接口命名
为满足装配规划中的约束需求，需定义以下接口：
*   **轴接口**：
    *   `central_axis`：通过点 $(0,0,0)$ 和 $(0,0,0.02)$ 创建，作为旋转中心轴。
*   **面接口**：
    *   `mid_plane`：基于 `XY` 平面向上偏移 $10\text{mm}$ ($0.01\text{m}$)，作为轴向对齐的中轴面。
    *   `front_face`：基于 `XY` 平面向上偏移 $20\text{mm}$ ($0.02\text{m}$)，作为轴向定位参考面。
    *   `bottom_face`：即 `XY` 平面（原点所在面）。

### 5. 风险点说明
*   **单位换算**：封装 API 要求使用 **米 (m)**。输入参数为 mm，代码中必须严格执行 `/1000` 换算（如 20mm -> 0.02m）。
*   **标准件物化**：必须先调用 `copy_standard_part_to_workdir_and_open`，确保不污染原始模板。
*   **中面对齐**：装配规划强调“所有零件轴向中点对齐于全局XY平面”，因此 `mid_plane` 的准确性至关重要。由于模板是从 $Z=0$ 开始拉伸的，中面应位于 $Z = B/2$ 处。
