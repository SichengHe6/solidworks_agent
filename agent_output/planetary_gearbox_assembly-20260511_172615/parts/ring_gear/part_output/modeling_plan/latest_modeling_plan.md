## 零件建模思路：内齿圈 (Ring Gear)

该零件是行星齿轮减速机构的固定基准，采用内齿环结构。为了保证装配精度和简化建模，将基于标准齿轮参数进行参数化建模。

### 1. 特征顺序规划

1.  **创建基础环体**：
    *   在 `XY` 平面绘制两个同心圆：外径 $120\text{ mm}$，内径（齿顶圆直径）约为 $104\text{ mm}$（计算值：$M \times (Z - 2 \times Hax) = 2 \times (54 - 2) = 104\text{ mm}$，此处需注意内齿轮齿顶圆在内侧）。
    *   使用**两侧对称**拉伸，总深度 $20\text{ mm}$。
2.  **生成内齿轮特征**：
    *   由于当前 API 封装主要支持外齿轮标准件修改，内齿圈将通过在环体内壁进行“拉伸切除”或调用特定内齿逻辑实现。
    *   **策略选择**：在环体内孔表面，通过草图绘制单个齿槽形状，然后进行圆周阵列切除（54次）。
3.  **创建参考几何体（接口）**：
    *   创建中心基准轴。
    *   创建中面基准面（用于轴向对齐）。

### 2. 关键草图与参数

*   **草图1 (Base Ring)**：位于 `XY` 平面。
    *   外圆：$\phi 120\text{ mm}$
    *   内圆（齿顶圆）：$\phi 104\text{ mm}$ (基于模数2, 齿数54计算)
*   **草图2 (Tooth Slot)**：位于环体端面。
    *   绘制符合模数 2 的渐开线齿槽形状（或简化梯形齿槽用于示意，若 API 支持标准件则优先使用标准件逻辑）。
*   **拉伸参数**：
    *   深度：$0.02\text{ m}$ ($20\text{ mm}$)
    *   方式：Mid Plane（两侧对称），确保 `XY` 平面始终处于齿宽中点。

### 3. 接口命名规范

*   **轴接口**：
    *   `central_axis`：位于 $(0,0,0)$ 到 $(0,0,0.02)$ 的中心轴，用于与太阳轮和行星架同轴约束。
*   **面接口**：
    *   `mid_plane`：位于 $Z=0$ 的参考平面（由 `XY` 平面偏移 0 得到），用于所有齿轮的轴向对齐。
    *   `inner_cylindrical_face`：内齿分度圆所在虚拟面（用于装配检查）。

### 5. 风险点说明

1.  **内齿计算**：内齿轮的齿顶圆直径小于分度圆直径（$D_a = M \times (Z - 2)$），与外齿轮相反，建模时需严格校验尺寸，避免与行星轮干涉。
2.  **单位换算**：API 要求使用 **米 (m)**，所有 $20\text{ mm}$、$120\text{ mm}$ 等尺寸必须除以 1000。
3.  **性能风险**：若使用草图阵列 54 个齿形，在大规模装配中可能导致卡顿，优先考虑特征阵列。
4.  **中面对齐**：必须确保拉伸方式为“两侧对称”，否则 `mid_plane` 与 `XY` 平面重合的假设将失效，导致装配错位。
