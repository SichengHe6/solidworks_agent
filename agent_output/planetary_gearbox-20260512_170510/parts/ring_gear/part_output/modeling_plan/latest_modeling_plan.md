## 零件建模思路：内齿圈 (Ring Gear)

### 1. 基础实体与主坐标系
*   **主坐标系**：以齿圈几何中心为原点 (0,0,0)，Z轴为旋转中心轴。
*   **建模基准**：
    *   `XY Plane`：作为齿圈的底面基准（Z=0）。
    *   `Front Plane (XY)`：用于创建外径和内径草图。
*   **单位**：所有尺寸在代码实现时将从 mm 转换为米 (m)。

### 2. 特征建模顺序
1.  **物化标准件**：从标准件库 `spur_gear.SLDPRT` 复制并重命名为 `ring_gear.SLDPRT`。
2.  **参数驱动修改**：
    *   设置模数 `M = 2`。
    *   设置齿数 `Z = 54`。
    *   设置齿宽 `B = 10` (0.01m)。
    *   *注意：标准件通常生成外齿轮，内齿圈需要通过“外圆环 - 齿形切除”或特定的内齿模板实现。根据 `gear_standard_part_workflow`，我们将基于标准件的主动参数生成基础齿形，并进行二次加工。*
3.  **外圆环特征**：
    *   在 `XY` 平面创建草图，绘制直径为 `OD = 120mm` (0.12m) 的圆。
    *   向上拉伸 `10mm` (0.01m)，确保与齿轮宽度一致。
4.  **内齿特征实现**：
    *   由于标准件模板通常为外齿，本零件将采用“反向逻辑”：先创建一个外径 120mm 的实体，再利用标准件生成的齿形参数进行内孔切除，或者直接调整模板参数。
    *   *关键假设*：如果标准件不支持直接切换为内齿，将通过拉伸一个足够大的外圆环，并减去（切除）标准齿轮生成的空间来模拟内齿效果。

### 3. 关键草图与基准
*   **Sketch_Outer_Boundary**：位于 `XY` 平面，定义外径 120mm。
*   **Sketch_Mounting_Holes**（如有）：本设计暂不包含额外安装孔，仅保留外圆。
*   **Reference Plane**：在 `Z = 0.01m` 处创建 `top_plane`。

### 4. 接口命名
为确保装配体能够正确引用，定义以下稳定接口：
*   **面接口**：
    *   `mounting_face`：位于 `Z = 0` 的底面，用于轴向定位。
    *   `front_face`：位于 `Z = 0.01` 的顶面。
*   **轴接口**：
    *   `central_axis`：通过 (0,0,0) 和 (0,0,0.01) 创建的中心旋转轴。
*   **点接口**：
    *   `center_point`：原点 (0,0,0)。

### 6. 风险点与注意事项
*   **内齿逻辑**：标准件 `spur_gear.SLDPRT` 若仅支持外齿，需确保切除逻辑正确，使留下的齿形符合 `M=2, Z=54` 的内啮合要求。
*   **干涉预警**：内齿圈的齿顶圆直径应小于行星轮的公转轨道边界，需校验 `OD=120` 是否有足够的壁厚。
*   **坐标系对齐**：必须确保 `mounting_face` 严格位于 `Z=0`，以符合装配规划中的“一键对齐”要求。
