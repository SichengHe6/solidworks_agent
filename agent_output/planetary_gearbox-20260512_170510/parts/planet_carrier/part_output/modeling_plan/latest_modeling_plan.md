# 行星架 (Planet Carrier) 建模思路

该零件为行星减速器的核心支撑部件，采用三叉式盘状结构，负责定位三个行星轮并输出扭矩。

## 1. 建模环境与基准设置
- **单位系统**：米 (m)。代码中将 mm 转换为 m（如 8mm -> 0.008m）。
- **主基准**：以 XY 平面为基准面，原点 (0,0,0) 作为行星架的旋转中心。
- **坐标系意图**：Z 轴为旋转中心轴，XY 平面为底面支撑参考。

## 2. 特征建模顺序
1.  **主体圆盘建模**：
    - 在 **XY 平面** 创建草图。
    - 绘制以原点为中心的圆，直径需略大于销轴分布圆直径（建议直径 90mm，即半径 0.045m，以保证结构强度）。
    - 向 **+Z 方向** 拉伸，深度为 `Thickness = 8mm` (0.008m)。
2.  **行星轮销轴建模**：
    - 在圆盘顶面（Z=0.008m）创建草图。
    - 绘制三个直径为 `Pin_Dia = 10mm` (0.01m) 的圆。
    - **分布位置**：位于直径 `Pin_Circle_Diameter = 72mm` (半径 0.036m) 的圆周上，角度分别为 0°、120°、240°。
        - 销轴1中心：(0.036, 0)
        - 销轴2中心：(-0.018, 0.03118)
        - 销轴3中心：(-0.018, -0.03118)
    - 向 **+Z 方向** 拉伸，拉伸高度应匹配行星轮宽度（10mm）及必要间隙，此处设为 12mm (0.012m)。
3.  **中心输出轴（可选/示意）**：
    - 若需体现输出接口，可在圆盘底面（Z=0）向 **-Z 方向** 拉伸一小段圆柱作为输出轴接口。

## 3. 接口命名规划
为方便装配体调用，需创建以下命名参考几何体：
- **面接口**：
    - `base_face`：圆盘底面（Z=0 平面），用于轴向定位。
- **轴接口**：
    - `output_axis`：通过原点垂直于 XY 平面的中心轴。
    - `pin_1_axis`：1号销轴的中心轴线。
    - `pin_2_axis`：2号销轴的中心轴线。
    - `pin_3_axis`：3号销轴的中心轴线。
- **点接口**：
    - `center_point`：原点 (0,0,0)。

## 5. 风险点与注意事项
- **销轴坐标精度**：120度分布的坐标计算（$\cos 120^\circ, \sin 120^\circ$）需精确，否则会导致行星轮与内齿圈/太阳轮啮合干涉。
- **销轴高度**：销轴拉伸高度必须大于行星轮宽度（10mm），确保行星轮有足够的安装空间。
- **接口一致性**：销轴轴线的命名必须与装配规划中的 `pin_n_axis` 严格一致，否则装配脚本将无法识别。
