# 太阳轮 (sun_gear) 建模思路

该零件为行星减速机构的输入中心齿轮，采用标准件驱动模式建模，并进行二次特征加工。

## 1. 基础基准与坐标系
*   **原点**：位于齿轮几何中心，对应装配体中的太阳轮旋转中心。
*   **基准面**：
    *   `XY平面`：齿轮端面参考面，齿轮沿 Z 轴正向拉伸。
    *   `XZ/ZY平面`：对称中心面。
*   **单位**：所有输入尺寸（mm）在代码实现时将转换为米（m）。

## 2. 特征建模顺序
1.  **物化标准件**：
    *   从 `D:\a_src\python\sw_agent\standard_swpart\gear\spur_gear.SLDPRT` 复制并重命名为 `sun_gear.SLDPRT` 到工作目录。
2.  **驱动参数设置**：
    *   修改全局变量 `M` (模数) = 2。
    *   修改全局变量 `Z` (齿数) = 18。
    *   修改全局变量 `B` (齿宽) = 10。
3.  **二次特征 - 中心通孔**：
    *   在 `XY平面`（或齿轮底面）创建草图。
    *   绘制直径为 `12mm` 的圆，圆心对齐原点。
    *   执行 `extrude_cut`（拉伸切除），深度为 `10mm`（贯穿齿宽）。
4.  **接口创建**：
    *   创建参考轴和参考面以供装配调用。

## 3. 关键接口命名 (Interfaces)
根据装配规划要求，暴露以下稳定接口：
*   **轴接口**：
    *   `rotation_axis`：通过点 (0,0,0) 和 (0,0,0.01) 创建的中心旋转轴。
*   **面接口**：
    *   `front_face`：位于 Z=0.01m 处的齿轮顶面（或通过 `XY` 平面偏移 `B` 得到）。
    *   `back_face`：位于 Z=0 处的齿轮底面（即 `XY` 平面）。
*   **点接口**：
    *   `center_point`：坐标 (0,0,0)，作为几何中心参考。

## 5. 风险点与注意事项
*   **单位换算**：API 内部使用米，12mm 需输入 `0.012`，10mm 需输入 `0.01`。
*   **标准件依赖**：必须确保标准件模板路径正确且全局变量名称（M, Z, B）与模板匹配。
*   **轴向一致性**：根据评审建议，`front_face` 需确保在装配时能与行星轮正确对齐，建模时统一从 Z=0 向上拉伸。
*   **干涉预警**：Z=18 为较小齿数，需确保齿顶高系数 Hax=1.0 以维持标准齿形，避免装配后与行星轮产生根切或干涉。
