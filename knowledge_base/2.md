# 装配规划智能体知识库 (Planning & Architecture)

## 1. 核心任务
你的任务是根据“需求概念说明书”，结合本库的 API 签名，制定一套严谨的**工程实现方案**。你必须明确指定每个零件使用的基准面、创建的装配轴点位、以及装配时的确切 API 调用逻辑。

## 2. 核心 API 签名 (API Dictionary)
规划时，请严格参考以下 API 签名进行节点设计：

### 2.1 空间与文档基建
* `create_workdir(assem_name)`: 创建工作目录。
* `get_part_path(workdir, part_name)` / `get_assembly_path(workdir, assem_name)`: 获取路径。
* `PartDoc(sw.createAndActivate_sw_part(name))` / `AssemDoc(sw.createAndActivate_sw_assembly(name))`

### 2.2 零件基准构建 (极度重要)
* **创建装配轴**: `create_axis(pt1: tuple, pt2: tuple, axis_name: str)`
  * *规划规则*: 所有的装配轴方向 (`pt1 -> pt2`) 必须在全局统一定义（例如：所有竖直孔/轴一律为 `-Y -> +Y`）。
* **主力工作平面**: `create_workplane_p_d(plane, offset_val: float)`
  * *规划规则*: 用于常规的特征叠加（如基于 XY 偏移 0.01m 画草图）。
* **命名装配面**: `create_ref_plane(plane, offset_val: float, target_plane_name: str)`
  * *规划规则*: 当需要为后续的平面装配留出接口时，**必须**使用此接口创建带有特定名称的平面（如 `target_plane_name="MatePlane_Top"`）。

### 2.3 零件特征构建 (用于梳理建模顺序)
* `insert_sketch_on_plane(plane)`: `plane` 必须与后续草图几何的 `sketch_ref` 严格一致（如 "XY", "XZ", "ZY" 或创建的基准面）。
* `extrude(...)` / `extrude_cut(...)` / `revolve(...)` / `revolve_cut(...)` / `sweep(...)` / `shell(...)` / `fillet(...)` / `chamfer(...)`
  * *规划规则*: 你需要为每个特征指定一个清晰的顺序，并且确保每个特征的草图基准面与前面创建的基准面或装配轴保持一致。

### 2.4 组件装配逻辑
* **导入组件**: `add_component(file_path, x, y, z) -> str` (返回组件名 comp_name)
* **同轴配合**: `mate_axes(assem_name, comp1, comp2, axis_name1, axis_name2, aligned: bool)`
* **平面配合**: `mate_faces(assem_name, comp1, comp2, plane_name1, plane_name2, aligned: bool)`
  * *规划规则*: `plane_name` 必须是两个零件中共有的内建面（如 `"前视基准面"`）或通过 `create_ref_plane` 预先创建的同名面。

## 3. 高级装配设计模式 (Advanced Assembly Patterns)
为了保证自动化装配的绝对稳定，你必须在规划中发散并应用以下核心思想：

### 3.1 双轴绝对防转模式 (Dual-Axis Anti-Rotation)
* **应用场景**：当两个零件需要组合且**不能发生相对旋转**时（如方形盖板、非圆柱形插槽）。
* **规划策略**：不要依赖复杂的侧面重合配合！必须在两个零件的相同相对位置，各预埋**两根**方向一致的基准轴。
  * 轴1：`Axis_Main`（通常在中心，用于锁定 X 和 Z 的平移）。
  * 轴2：`Axis_Edge`（偏置一定距离，如位于某条棱上，用于彻底锁死绕 Y 轴的旋转）。
* **装配执行**：连续调用两次 `mate_axes`，将平移和旋转自由度进行数学级锁定。

### 3.2 坐标系解耦与虚拟装配面 (Virtual Mating Planes)
* **规划策略**：彻底抛弃“找物理面”的思路。利用 `create_ref_plane` 在绝对坐标高度预埋**虚拟接触面**。
  * *范例*：要在深度 5mm 的沉头孔底部放螺丝，无需管切除特征怎么成型，直接在孔底绝对高度（如 Y=0.025）处 `create_ref_plane(..., target_plane_name="Mate_Hole_Bottom")`。
* **装配执行**：使用 `mate_faces` 直接调用该命名虚拟面，实现与几何拓扑完全解耦的高鲁棒性配合。

### 3.3 全局向量一致性协议 (Global Vector Protocol)
* **应用场景**：所有涉及 `mate_axes` 的轴配合。
* **规划策略**：不管零件是正着画还是倒着画，其装配轴的起点 (`pt1`) 到终点 (`pt2`) 的向量方向必须在整个装配体中保持绝对一致。
  * *范例*：规定所有竖直连接部件（桌腿、螺丝、销钉）建轴时一律从底向顶 `(0, -Y, 0) -> (0, +Y, 0)`。
* **装配执行**：这样可以闭眼将所有 `mate_axes` 的 `aligned` 参数设为 `True`，永久消灭零件装配后翻转/倒置的 Bug。
