# SolidWorks 能力边界

该文档由 v2 `RequirementAgent Knowledge Base` 归纳而来，用于需求补全阶段判断系统能做什么、不能承诺什么。

## 稳定支持

- 单零件参数化建模。
- 多零件常规装配。
- 基于二维草图的建模：
  - `extrude`
  - `extrude_cut`
  - `revolve`
  - `revolve_cut`
  - `sweep`
- 常见草图元素：
  - 圆
  - 中心矩形
  - 多边形
  - 折线/线段
  - 三点圆弧
  - 椭圆
  - 开放方程样条
  - 草图圆角
  - 草图倒角
- 常见三维特征：
  - `shell`
  - 边/面圆角
  - 边/面倒角
- 基准几何：
  - 平行偏移基准面
  - 两点定义基准轴
- 装配：
  - 插入零件
  - 平面配合
  - 轴同心配合

## 不应承诺或需要降级表达

- 自由曲面、复杂外形曲面、`Loft`、`Boundary`、`Blend`。
- 柔性机构和高级运动仿真。
- 高级装配约束，例如宽度、路径、限位等。
- 复杂高级孔特征库。
- 高风险复杂布尔流程。
- 不稳定或未正式封装的高级阵列能力。
- 闭合方程曲线。

## 需求补全时的表达方式

如果用户提出能力边界外的需求，应尽量转写为当前封装可实现的结构：

- 把复杂外形近似为拉伸、旋转、切除和圆角/倒角组合。
- 把装配关系落到平面重合、轴同心和固定基准件。
- 把运动关系写成装配意图或预留接口，不承诺高级仿真。
- 把模糊结构转换为主体、草图面、关键尺寸、孔槽、接口名和保存目标。
