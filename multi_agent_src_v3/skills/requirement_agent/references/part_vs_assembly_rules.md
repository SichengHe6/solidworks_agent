# 零件与装配体判定规则

- 单个连续建模实体或单个可复用组件使用 `part`。
- 当请求提到多个组件、重复实例、运动、约束、配合、轴/齿轮/轴承、紧固件或相对位置时，使用 `assembly`。
- 装配体中的相同重复对象应作为一个唯一零件建模，再通过多个实例表达。
- 当交付物包含 `.SLDPRT` 或只描述一个实体对象时，倾向 `part`。
- 当交付物包含 `.SLDASM`，或用户要求“装起来”“组装”“配合”时，倾向 `assembly`。
- 多个几何相同但接口用途不同的实例，仍优先复用同一个 `part`，在 `instances[].interface_usage` 中区分实际使用接口。
