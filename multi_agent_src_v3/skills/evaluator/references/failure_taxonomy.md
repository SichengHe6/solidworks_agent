# 失败分类

- json_parse_error
- requirement_completion_error
- part_modeling_plan_error
- part_code_implementation_error
- asset_binding_error
- static_validation_error
- solidworks_execution_error
- screenshot_error
- geometry_feature_missing
- geometry_topology_invalid
- assembly_plan_error
- assembly_code_error
- constraint_error
- asset_interface_error
- kinematic_validation_error
- unknown

零件修复报告应映射到 `modeling_plan_error`、`code_implementation_error`、`asset_binding_error` 或 `unknown`。

装配修复报告应映射到 `assembly_plan_error`、`assembly_code_error`、`constraint_error`、`asset_interface_error` 或 `unknown`。
